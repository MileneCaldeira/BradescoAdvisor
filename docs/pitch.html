01 — O Problema

O relacionamento bancário
ainda é genérico e lento

Clientes do Bradesco enfrentam dificuldades para tirar dúvidas financeiras rapidamente, entender produtos complexos e simular cenários — dependendo de atendimento humano ou FAQs estáticos.

⏱️

Atendimento lento

Tempo médio de espera para dúvidas simples ultrapassa 15 minutos em centrais telefônicas.

📄

FAQs estáticos

Respostas genéricas que não consideram o contexto do cliente nem permitem simulações.

💸

Custo operacional

Alto custo de atendimento humano para perguntas repetitivas que poderiam ser automatizadas.

02 — A Solução

IA Generativa aplicada ao
relacionamento financeiro

O BradescoAdvisor combina linguagem natural, base de conhecimento estruturada e simulações matemáticas reais para oferecer uma experiência de alto valor ao cliente.

FEATURE 01
🧠

FAQ Inteligente

Respostas contextualizadas sobre produtos, processos e serviços Bradesco — com memória de conversa e personalização por nome.

FEATURE 02
📊

Simulações Financeiras

Cálculos reais de empréstimos (Tabela Price) e investimentos (juros compostos + IR regressivo) com tabela de amortização.

FEATURE 03
🔐

Guardrails de Segurança

Disclaimers automáticos, sem captação de dados sensíveis, orientação responsável e respeito aos limites regulatórios.

FEATURE 04
📈

Avaliação Contínua

Sistema de métricas automáticas em 5 dimensões para medir e melhorar a qualidade das respostas em produção.

03 — As 6 Etapas

Desenvolvimento estruturado,
etapa a etapa

Cada etapa do desafio foi implementada como um módulo técnico independente, garantindo qualidade e rastreabilidade.

Etapa 1

Documentação do Agente

Especificação técnica completa: objetivos, casos de uso, arquitetura, guardrails, métricas de sucesso e limitações conhecidas. Arquivo: docs/agent_documentation.md

Etapa 2

Base de Conhecimento

JSON estruturado com produtos Bradesco (contas, cartões, investimentos, crédito, seguros), FAQ com 8 perguntas e taxas de mercado. Arquivo: data/knowledge_base.json

Etapa 3

Prompts do Agente

3 prompts engenheirados: system prompt com identidade e guardrails, intent classifier com output JSON, e simulation prompt com chain-of-thought. Arquivo: prompts/agent_prompts.py

Etapa 4

Aplicação Funcional

App Streamlit completo com chat, KPIs, quick buttons, métricas inline. Engine local de simulações financeiras (Price + Compostos + IR). Arquivos: app.py + utils/financial_calculator.py

Etapa 5

Avaliação e Métricas

Sistema de avaliação com 5 dimensões (Clarity, Domain Relevance, Completeness, Helpfulness, Safety) + score composto + relatório de recomendações. Arquivo: evaluation/metrics.py

Etapa 6

Pitch e Entrega

Pitch deck profissional em HTML com identidade visual Bradesco, README de nível sênior, e estrutura de repositório pronta para GitHub. Este documento.

04 — Arquitetura Técnica

Decisões de design
orientadas a produção

O agente foi construído com separação de responsabilidades, orquestração explícita e módulos reutilizáveis — pronto para escalar.

USUÁRIO → Streamlit UI
             ↓
ORQUESTRAÇÃO (app.py)
    ├── 1. Intent Detection   → Claude API (max 200 tokens)
    ├── 2. Simulation Check  → Local Python (sem latência)
    ├── 3. Context Builder   → KB (3000 chars) + History (8 msgs)
    ├── 4. Main Chat        → Claude API (max 1500 tokens)
    └── 5. Metrics Logging  → AgentEvaluator
             ↓
RESPOSTA → Chat UI + Badge de Intenção + Painel de Métricas

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CAMADAS DE SUPORTE
  knowledge_base.json    → Produtos, FAQ, Taxas
  agent_prompts.py       → System, Intent, Simulation
  financial_calculator.py → Price, Compostos, IR Regressivo
  evaluation/metrics.py   → 5 dimensões + Score Composto
05 — Avaliação

Métricas objetivas para
melhoria contínua

O sistema de avaliação mede qualidade em 5 dimensões, gera scores compostos e recomendações acionáveis — sem depender de avaliação manual.

25%
Clarity
(estrutura + markdown)
25%
Domain Relevance
(termos financeiros)
20%
Completeness
(tamanho adequado)
30%
Helpfulness + Safety
(orientação responsável)

🎯 Avaliação automática

Score composto calculado por heurísticas auditáveis, independente do modelo — reproduzível e sem custo adicional de API.

💡 Recomendações acionáveis

O sistema gera sugestões concretas baseadas nos scores: quando revisar prompts, otimizar latência ou expandir a base de conhecimento.
